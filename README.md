# YZ Meuble

Site officiel : https://renolix.github.io/yzmeuble.github.io/

Site vitrine React pour YZ Meuble : catalogue sans prix, fiches produit,
présentation du sur-mesure et formulaires de devis connectés à WhatsApp.

## Lancer le projet

```bash
npm install
npm run dev
```

## Créer la version de production

```bash
npm run build
```

Les fichiers prêts pour GitHub Pages sont générés dans `docs`.

## Personnalisation

- Les produits et collections se trouvent dans `src/data.js`.
- Le numéro WhatsApp est défini dans `src/App.jsx`.
- Les styles complets se trouvent dans `src/styles.css`.
